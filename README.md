This is the repo for the GraphQL by PoP site: https://graphql-by-pop.com

## About this repo

This repo is a clone from [Hylia](https://github.com/hankchizljaw/hylia) (a starter kit based on the [Eleventy](https://11ty.io/) static site generator). Please check out this project for additional configuration information, concerning design tokens, styleguide, and others.

## Install

1. Clone or fork this repo: `git clone https://github.com/getpop/graphql-by-pop.com`
2. `cd` into the project directory and run `npm install`
3. Once all the dependencies are installed run `npm start`
4. Open your browser at `http://localhost:8080` and away you go!

## Terminal commands

### Serve the site locally

```bash
npm start
```

### Build a production version of the site

```bash
npm run production
```

### Compile Sass

```bash
npm run sass:process
```

### Re-generate design tokens for Sass

```bash
npm run sass:tokens
```
